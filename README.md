# Spotify Lyrics Overlay for Windows

An open-source Windows widget that displays real-time Spotify lyrics in a floating, always-on-top overlay.

This project is in early planning stage.

## Goals
- Floating, movable lyrics overlay
- Always-on-top window
- Full customization (opacity, themes, fonts)
- Accessibility-first design
- Lightweight and privacy-respecting
